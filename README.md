# Azure Cloud Ops

A collection of notes, blog posts, and practical artifacts (Terraform/Bicep/scripts) focused on operating Azure environments—especially Azure Government / regulated patterns.

## What’s in this repo
- `_posts/`: Jekyll blog posts
- `_pages/`: site pages
- `bicep/`: Bicep templates and examples
- `terraform/`: Terraform examples
- `powershell/`, `bash-scripts/`: helper scripts
